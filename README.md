# omega (๑•̀ω•́ ๑)

[![License](https://img.shields.io/badge/license-BSD-blue.svg)](LICENSE)

#### 介绍
只包含头文件的常用模块库。

#### 软件架构
软件需要标注 `C++11` 支持，对应 `VS2015` 或更高版本，`GCC 5.0` 或更高版本。


#### 安装教程

1.  将 `include/ohm` 文件夹拷贝到项目的头文件目录即可。

#### 使用说明

1.  查看 `ctest` 和 `test` 获取更多的示例。
2.  请耐心等待项目编译。
3.  如果等待太烦人了，建议只在不经常更新的源文件中使用 `ohm`。
